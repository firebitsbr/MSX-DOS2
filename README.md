## MSX-DOS2

日本語 MSX-DOS2 Version 2.20〜2.31 をベースに記載していますが、<br />
エミュレータ等の独特な動作に関しても記載を行っているコマンドがあります。

.txt ファイルは文字コード Shift_JIS、改行コード CR+LF で記載しています。<br />
MSX のテキストエディタも使用できますが、一部ファイル名の変更が必要です。<br />
Windows では メモ帳 を使用可能です。<br />
