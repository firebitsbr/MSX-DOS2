BASIC
-----------------------------------------------------------
内部コマンド

[ 機能 ]
MSX-Disk BASIC に制御を移します。

[ 書式 ]
BASIC [<ファイル名>]

[ 解説 ]
<ファイル名> を入れると、
BASIC 起動時にそのファイルが読み込まれ、実行されます。
BASIC のみの場合は起動のみが行なわれます。
BASIC から MSX-DOS に戻る場合は
CALL SYSTEM コマンド(_SYSTEM)を用います。
CALL SYSTEM(<コマンド>) によるコマンド指定がある場合は
そのコマンドを実行しますが、
省略されている場合はバッチファイル REBOOT.BAT を探し、
存在している場合は実行されます。
