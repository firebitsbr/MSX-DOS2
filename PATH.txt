PATH
-----------------------------------------------------------
内部コマンド

[ 機能 ]
COM ファイル・BAT ファイルの検索パスを設定・表示します。

[ 書式 ]
PATH [[+|-] <パス> [<パス> [<パス> ... ]]]

[ 解説 ]
PATH のみの場合は現在の設定を表示します。区切りは ;（セミコロン）です。

- を加えた場合は、現在の設定から <パス> を削除します。

+ を加えた場合は、現在の設定に <パス> が追加されます。既に存在している場合は削除され、後ろに追加されます。これにより検索順序を変更する事ができます。

- および + を加えずに <パス> を記載している場合は、現在の設定が削除され、記載された設定となります。

検索パスの設定は環境変数 PATH に入るため、SET コマンドを用いて変更を行っても構いません。
