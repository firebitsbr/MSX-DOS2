IF
-----------------------------------------------------------
内部コマンド
2.31 以降対応

[ 機能 ]
条件判断に応じてコマンドを実行します。

[ 書式 ]
IF [NOT] <条件> <コマンド>

[ 解説 ]
<条件> が正しい場合に <コマンド> を実行します。
NOT を指定した場合は正しくない場合に <コマンド> を実行します。
<条件> として、以下が使用できます。

	EXIST <ファイル名>
		<ファイル名> が存在する場合にコマンドを実行します。

	<文字列1>==<文字列2>
		<文字列1> <文字列2> が等しい場合にコマンドを実行します。
		<文字列1> <文字列2> はパラメータと環境変数を指定する事ができ、
		これらを比較する事ができます。

IF コマンドは version 2.31 で追加されました。
